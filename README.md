## Project Overview
This project analyzes regional sales data for ABC Products to evaluate sales performance
across different regions and product categories. The goal is to provide management with
actionable insights using Excel-based analysis and visualization techniques.

## Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Excel Formulas
- Charts & Dashboards

## Dataset Description
- Format: CSV
- Records: ~4000 sales transactions
- Key Columns: Order Date, Region, Product Category, Sales Amount, Discount, Profit

## Key Tasks Performed
- Data cleaning and text standardization using Excel functions
- Filtering sales by region, category, and time period
- Calculating total sales, average discount, and profit
- Creating Pivot Tables for regional and category-level analysis
- Performing basic regression analysis between discount and sales
- Building an interactive Excel dashboard

## Key Insights
- South region shows strong performance in Electronics category
- Higher discounts do not always lead to higher profits
- Certain product categories consistently outperform others

This project demonstrates practical Excel skills including data cleaning, analysis,
visualization, and dashboard creation for business decision-making.
